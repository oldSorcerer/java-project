
### Отчёт об отличниках в Университете Просвещённых Умов ✨

Вы — декан самой престижной кафедры в "Университете Просвещённых Умов", и вам поручено выявить самых ярких студентов.

Сначала создайте класс Student с полями studentName, studyCourse, studentSpecialty и averageGrade.

Затем у вас есть полный реестр учащихся (создайте список студентов с разными курсами, специальностями и баллами).

Ваша задача — используя Stream API, создать подробный отчёт.

Вам нужно сгруппировать студентов сначала по их studyCourse, а затем внутри каждого курса — по studentSpecialty.
В каждой такой подгруппе вам предстоит подсчитать, сколько студентов демонстрируют выдающиеся успехи, то есть имеют averageGrade больше 4.0.
Конечный результат должен быть представлен в виде Map<Integer, Map<String, Long>>, где ключи — это курс и специальность, а значение — количество отличников.

Когда отчёт будет готов, выведите его на экран: для каждого курса и каждой специальности укажите, сколько студентов показали блестящие результаты.

#### Требования:
1. Необходимо создать класс Student с полями studentName (имя студента), studyCourse (курс), studentSpecialty (специальность) и averageGrade (средний балл).
2. Необходимо создать список объектов Student, в котором студенты имеют различные курсы, специальности и средние баллы.
3. Необходимо сгруппировать студентов сначала по полю studyCourse (курс), затем внутри каждой группы — по полю studentSpecialty (специальность).
4. В каждой подгруппе (по курсу и специальности) нужно подсчитать количество студентов, у которых averageGrade больше 4.0.
5. Итоговая структура данных должна быть Map<Integer, Map<String, Long>>, где Integer — курс, String — специальность, Long — количество отличников.
6. Необходимо вывести на экран для каждого курса и каждой специальности количество студентов, средний балл которых превышает 4.0.
