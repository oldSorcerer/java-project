
### Запуск сердца приложения через Maven ✨

Ваш модуль app теперь содержит бьющееся сердце вашего приложения — главный класс com.example.app.Main, который запускает всю магию. Но что толку от сердца, если его трудно запустить? Вы хотите сделать так, чтобы любой разработчик или даже автоматизированная система сборки могли мгновенно вдохнуть жизнь в ваше приложение, используя всего лишь одну простую и запоминающуюся команду Maven.

Ваша миссия — вдохнуть эту возможность в pom.xml файл модуля app. Вам предстоит настроить специальный плагин, exec-maven-plugin, таким образом, чтобы, когда кто-либо выполнит команду mvn -pl app exec:java, ваш класс com.example.app.Main немедленно ожил и начал выполнять свою работу. Это позволит вашему приложению легко и элегантно запускаться, словно по волшебству.

#### Требования:
1. В pom.xml модуля app должен быть добавлен exec-maven-plugin.
2. В конфигурации exec-maven-plugin должен быть явно указан основной класс com.example.app.Main.
3. Настройка должна позволять запускать приложение командой mvn -pl app exec:java без дополнительных параметров.
4. exec-maven-plugin должен быть добавлен в секцию <build><plugins> pom.xml модуля app.
5. В конфигурации плагина не должно быть лишних или конфликтующих параметров, мешающих запуску основного класса.
6. Настройка должна находиться только в pom.xml модуля app, а не в корневом pom.xml всего проекта.
