
### Умный конфигуратор для вашего приложения ⚙️

Вы создаете универсальный конфигуратор для своего приложения, который позволяет пользователю выбирать, как сохранять настройки. Пользователь может либо добавить новые настройки к существующим, либо полностью перезаписать их.

Ваша программа должна сначала запросить у пользователя режим работы: "append" (добавить) или "overwrite" (перезаписать). Затем она должна запросить сам текст настроек.

Если пользователь выбрал "append", ваш конфигуратор должен бережно добавить этот текст в конец файла "app_settings.conf", сохраняя все предыдущие настройки. Если же был выбран "overwrite", конфигуратор должен смело стереть все старые настройки и записать только новый текст.

В результате, файл "app_settings.conf" должен содержать только то, что соответствует выбранному режиму.

#### Требования:
1. Программа должна запросить у пользователя режим работы: "append" или "overwrite".
2. Программа должна запросить у пользователя текст новых настроек, который требуется сохранить.
3. Программа должна производить запись в файл с именем "app_settings.conf".
4. Если пользователь выбрал режим "append", программа должна добавить введённый текст в конец файла "app_settings.conf", не удаляя существующее содержимое.
5. Если пользователь выбрал режим "overwrite", программа должна удалить всё старое содержимое файла "app_settings.conf" и записать только введённый пользователем текст.
6. В результате выполнения программы файл "app_settings.conf" должен содержать только те данные, которые соответствуют выбранному пользователем режиму работы.
