
### Командный пункт запуска ракеты ☄️

Вы — ведущий разработчик сложнейшей системы управления запуском межпланетной ракеты. На главном пульте управления есть одна-единственная, но критически важная кнопка "Запуск". Когда эта кнопка нажата, она должна мгновенно уведомить несколько ключевых департаментов, и их системы должны отреагировать в строго определенном порядке, чтобы избежать хаоса.

Начните с построения центрального элемента вашего пульта — главного окна (JFrame) и разместите на нем большую, яркую кнопку JButton с надписью "Инициировать Запуск".

Теперь приступайте к созданию механизмов оповещения. Вам понадобятся два отдельных агента уведомлений (они же ActionListener). Первый агент, представляющий отдел Топливной Системы, при получении сигнала должен выводить в консоль сообщение: "Система Дозаправки: Проверка готовности завершена!". Второй агент, отвечающий за Навигационный Комплекс, должен сообщать: "Навигационная Система: Расчет курса успешно начат!".

Самое важное: прикрепите оба этих агента уведомлений к вашей кнопке "Инициировать Запуск". Убедитесь, что вы добавляете их последовательно: сначала Топливную Систему, затем Навигационную. При каждом нажатии на кнопку "Инициировать Запуск" в консоль должны выводиться оба сообщения, причем их порядок вывода должен в точности соответствовать порядку, в котором вы их добавили к кнопке. Это гарантирует, что последовательность запуска будет всегда предсказуемой и безопасной.

#### Требования:
1. В программе должен быть создан экземпляр JFrame, который является основным окном приложения.
2. На главном окне должен быть размещён компонент JButton с текстом "Инициировать Запуск".
3. Должны быть реализованы два отдельных слушателя событий (ActionListener): один для отдела Топливной Системы, второй для Навигационного Комплекса.
4. Первый ActionListener при срабатывании должен выводить в консоль сообщение: "Система Дозаправки: Проверка готовности завершена!".
5. Второй ActionListener при срабатывании должен выводить в консоль сообщение: "Навигационная Система: Расчет курса успешно начат!".
6. Оба ActionListener должны быть добавлены к кнопке "Инициировать Запуск" строго в порядке: сначала слушатель Топливной Системы, затем слушатель Навигационного Комплекса.
7. При каждом нажатии на кнопку оба сообщения должны выводиться в консоль, причём в том же порядке, в котором слушатели были добавлены к кнопке.
