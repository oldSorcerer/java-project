
### Магический импульс связи ✨

Вы — архимаг, отвечающий за создание новой системы связи между древними магическими артефактами. Каждый артефакт должен иметь возможность посылать "магический импульс", несущий в себе определённое послание. Ваша первостепенная задача — определить саму суть такого магического импульса.

Вам необходимо создать специальный класс, который будет представлять этот самый "магический импульс". Назовите его MagicalPulseEvent. Этот класс должен наследовать свою базовую природу от java.util.EventObject, что позволит ему гармонично вписаться в существующие магические протоколы. Каждый MagicalPulseEvent должен содержать в себе некое String сообщение, которое и является сутью передаваемой информации.

Реализуйте конструктор для MagicalPulseEvent, который будет принимать два аргумента: источник импульса (то есть, какой артефакт его породил) и само String сообщение. Также добавьте публичный метод getPulseMessage(), который позволит любому получателю извлечь это тайное послание.

Чтобы убедиться, что ваша концепция работает, создайте один такой MagicalPulseEvent, передав ему в качестве сообщения "Приветствие из Хрустальной Башни!". Затем используйте метод getPulseMessage() для извлечения и вывода этого сообщения на экран, демонстрируя успешность вашей магической разработки.

#### Требования:
1. Класс MagicalPulseEvent должен наследоваться от java.util.EventObject.
2. Класс MagicalPulseEvent должен содержать приватное поле типа String для хранения сообщения импульса.
3. Класс MagicalPulseEvent должен иметь публичный конструктор, принимающий два аргумента: Object source (источник события) и String message (сообщение импульса).
4. Класс MagicalPulseEvent должен предоставлять публичный метод getPulseMessage(), возвращающий сообщение импульса.
5. В программе должен быть создан экземпляр MagicalPulseEvent с произвольным объектом в качестве источника и строкой "Приветствие из Хрустальной Башни!" в качестве сообщения.
6. После создания события, программа должна вызвать метод getPulseMessage() и вывести возвращённое сообщение на экран.
