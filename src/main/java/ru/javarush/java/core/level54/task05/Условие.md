
### Измерение энергетического потенциала артефактов ✨

Вы — блестящий ученый, изучающий древние артефакты. У вас есть огромная коллекция из 1000 образцов, пронумерованных от 1 до 1000.

Чтобы разгадать их тайны, вам нужно вычислить их общий "энергетический потенциал", который определяется как сумма квадратов значений каждого артефакта.

Перед вами стоит гора данных, и чтобы быстро раскрыть ее секреты, вы должны использовать всю мощь параллельной обработки.

Создайте представление для вашей обширной коллекции артефактов и затем, используя параллельный поток, вычислите итоговую сумму их квадратов, чтобы показать этот окончательный показатель.

#### Требования:
1. Необходимо создать коллекцию, содержащую числа от 1 до 1000 включительно, где каждое число представляет отдельный артефакт.
2. Для обработки коллекции должен быть использован параллельный поток (parallel stream), а не обычный stream.
3. Для каждого элемента коллекции требуется вычислить квадрат его значения.
4. Необходимо вычислить сумму всех квадратов значений артефактов.
5. Программа должна вывести полученную сумму на экран.
