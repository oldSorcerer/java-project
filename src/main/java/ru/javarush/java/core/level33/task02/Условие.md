
### Анализ длины заклинаний для магической академии ✨

Вы – молодой ученик в Академии Тайных Знаний, и ваше сегодняшнее задание — проанализировать древние свитки с заклинаниями. Говорят, что чем длиннее имя заклинания, тем больше магической энергии оно в себе таит. Вам дан список названий заклинаний, и ваша цель — узнать длину каждого из них.

У вас есть список строк List<String> ancientSpells, содержащий названия заклинаний вроде "Java", "Stream", "API", "IntStream". Преобразуйте этот список в поток чисел (IntStream), где каждое число будет соответствовать длине одного из заклинаний. Затем, чтобы продемонстрировать свои магические способности, выведите длину каждого заклинания на новой строке. Это поможет вам понять, какие заклинания обладают наибольшим потенциалом.

#### Требования:
1. Необходимо использовать примитивный стрим IntStream для обработки длин строк, а не Stream<Integer>.
2. Каждое название заклинания из списка ancientSpells должно быть преобразовано в соответствующее целое число — его длину.
3. В процессе преобразования не должно происходить автоупаковки (boxing) строковых длин в объекты Integer — нужно работать с примитивными int.
4. Длина каждого заклинания должна быть выведена на отдельной строке в порядке следования в исходном списке.
