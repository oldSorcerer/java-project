
### Запуск беспилотного дрона-курьера ✈️

Представьте, что вы инженер, запускающий небольшой, но независимый дрон-курьер. Вы назвали его MessageDrone. Его единственная задача — взлететь, пролететь по маршруту и сообщить всем: "Hello from MessageDrone!".

Ваша цель — создать этот дрон как отдельный летательный аппарат, способный выполнять свои собственные инструкции, а затем дать ему команду на старт, чтобы он начал свою миссию и произнес свое приветствие из собственного независимого "потока выполнения".

#### Требования:
1. Необходимо создать отдельный класс с именем MessageDrone.
2. Класс MessageDrone должен реализовывать интерфейс Runnable для задания логики потока.
3. В классе MessageDrone необходимо переопределить метод run, который будет содержать инструкции для дрона.
4. Метод run должен выводить на экран строку "Hello from MessageDrone!".
5. В основной программе необходимо создать объект Thread, передав ему экземпляр MessageDrone.
6. Необходимо вызвать метод start() у созданного потока, чтобы запустить выполнение инструкций дрона в отдельном потоке.
