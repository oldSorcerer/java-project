
### Инженер на заводе

Представьте, что вы – главный инженер на заводе, который производит что-то очень важное, например, медицинское оборудование. Ваша задача – мгновенно сообщить операторам о любых нештатных ситуациях.

Ваша система должна использовать специальный журнал – java.util.logging.Logger – чтобы предупреждать о потенциальных проблемах и кричать "SOS!" в случае критического сбоя. Вы должны настроить программу так, чтобы она вывела два сообщения прямо в консоль. Первое сообщение – это деликатное "Внимание: что-то может пойти не так", которое будет уровнем WARNING, сигнализируя о необходимости повышенного внимания. Второе, гораздо более тревожное сообщение – "Произошла критическая ошибка", должно быть с уровнем SEVERE, указывая на немедленное вмешательство.

#### Требования:
1. В программе должен использоваться класс Logger из пакета java.util.logging для ведения журнала событий.
2. В программе должен быть получен экземпляр Logger с помощью метода Logger.getLogger().
3. Сообщения журнала должны выводиться непосредственно в консоль.
4. В консоль должно быть выведено сообщение "Внимание: что-то может пойти не так" с уровнем WARNING.
5. В консоль должно быть выведено сообщение "Произошла критическая ошибка" с уровнем SEVERE.
