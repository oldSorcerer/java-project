
### Социальная сеть

Как разработчик социальной сети, вы знаете, как важно понимать поведение пользователей. Отслеживать каждый их шаг — ключ к улучшению сервиса и выявлению проблем. Представьте, что вы хотите отследить, кто и какие действия совершил.

Ваша задача — использовать умное логирование, которое автоматически подставляет нужные детали, чтобы сообщение выглядело естественно и информативно. Используя SLF4J или Log4j, напишите код, который будет выводить в лог сообщение на уровне INFO. Шаблон сообщения должен быть "Пользователь {} выполнил действие {}", где вместо фигурных скобок будут автоматически подставлены значения переменных.

Вам нужно объявить две переменные, например, loggedInUser для имени пользователя и performedAction для описания его действия, и присвоить им произвольные значения. Убедитесь, что ваша программа выводит в лог итоговое сообщение с корректно подставленными значениями, что позволяет легко отслеживать активность.

#### Требования:
1. В программе должен быть импортирован и использован логгер SLF4J или Log4j, а также создан экземпляр логгера для текущего класса.
2. Должны быть объявлены две переменные, например, loggedInUser (имя пользователя) и performedAction (описание действия), которым присваиваются произвольные значения.
3. Для логирования должно использоваться сообщение-шаблон "Пользователь {} выполнил действие {}", где фигурные скобки предназначены для автоматической подстановки значений переменных.
4. Сообщение должно быть выведено именно на уровне INFO с помощью выбранного логгера.
5. Значения переменных loggedInUser и performedAction должны быть автоматически подставлены в шаблон сообщения без ручной конкатенации строк.
