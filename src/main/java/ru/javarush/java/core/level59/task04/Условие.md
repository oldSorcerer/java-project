
### Параллельная обработка частей одного файла

Представьте, что вы — агент спецслужб, и вам на перехват попала огромная, критически важная зашифрованная передача, записанная в файле intercepted_transmission.log. Этот файл настолько массивен, что расшифровывать его целиком в одиночку займет слишком много времени, а каждая секунда на счету! Вы решаете применить тактику разделения и параллельной обработки.

Ваш план состоит в следующем: вы делите эту колоссальную передачу на четыре идеально равные по размеру части. Затем для каждой части вы запускаете отдельного "агента-дешифровщика" (поток). Каждый такой агент должен использовать продвинутые методы прямого доступа к файлу, такие как FileChannel и ByteBuffer, чтобы быстро прочитать только свою назначенную часть сообщения.

Когда каждый дешифровщик завершит свою работу, он должен немедленно доложить: "Декодировано [количество_байт] байт, начиная с позиции [стартовая_позиция_в_файле]". Как только все четыре агента завершат свою миссию и отчитаются, ваша центральная система должна выдать финальное подтверждение: "Передача полностью расшифрована. Миссия успешно завершена!". Это обеспечит быструю и эффективную обработку критической информации.

#### Требования:
1. Для чтения частей файла должны использоваться классы FileChannel и ByteBuffer для прямого доступа к файлу.
2. Файл intercepted_transmission.log должен быть разделён на четыре равные по размеру части (по количеству байт).
3. Для обработки каждой части файла должен быть создан отдельный поток (агент-дешифровщик), всего четыре потока.
4. Каждый поток должен прочитать только свою часть файла, начиная с определённой стартовой позиции и ограничиваясь длиной своей части.
5. После завершения чтения своей части, каждый поток должен вывести сообщение: "Декодировано [количество_байт] байт, начиная с позиции [стартовая_позиция_в_файле]".
6. Основной поток должен дождаться завершения работы всех четырёх потоков-агентов.
7. После завершения всех потоков, программа должна вывести сообщение: "Передача полностью расшифрована. Миссия успешно завершена!".
