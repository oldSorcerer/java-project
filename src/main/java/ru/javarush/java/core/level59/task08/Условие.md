
### Параллельный подсчёт строк в большом файле

Вы — редактор в издательстве, и вам предстоит провести аудит огромной, многостраничной рукописи — текстового файла, содержащего миллионы строк. Вручную подсчитать количество строк — это работа на века! Чтобы справиться с этим колоссальным трудом, вы решаете применить гениальную стратегию: разделить рукопись на управляемые части и поручить подсчёт отдельной команде быстрых "счетоводов".

Ваша задача — разработать программу, которая сначала делит эту огромную рукопись на фрагменты, каждый размером не более 10 мегабайт. Для каждого такого фрагмента вы создаёте отдельное задание (можно использовать Runnable или Callable) для "счетовода", который специализируется на подсчёте строк (символов новой строки "\n") исключительно в своём фрагменте.

Для координации работы всей вашей команды "счетоводов" используйте ExecutorService. Запускайте каждое задание параллельно, чтобы они работали одновременно. Как только все ваши "счетоводы" завершат свою работу и передадут вам свои результаты, ваша программа должна суммировать их и с гордостью вывести на экран общее количество строк во всей рукописи. Это позволит вам в кратчайшие сроки получить точную информацию о масштабе произведения.

#### Требования:
1. Программа должна разбивать исходный файл на фрагменты размером не более 10 мегабайт каждый.
2. Для каждого фрагмента должен быть создан отдельный объект Runnable или Callable, который подсчитывает количество строк (символов новой строки \n) только в своём фрагменте.
3. Для выполнения задач по подсчёту строк необходимо использовать ExecutorService, чтобы запускать их параллельно.
4. После завершения всех задач программа должна корректно собрать результаты подсчёта строк из каждого задания и вычислить общее количество строк во всём файле.
5. Программа должна корректно обрабатывать ситуации, когда фрагменты делятся на середине строки, чтобы не допустить двойного или пропущенного подсчёта строк.
6. После завершения работы всех задач программа должна вывести на экран итоговое количество строк в исходном файле.
