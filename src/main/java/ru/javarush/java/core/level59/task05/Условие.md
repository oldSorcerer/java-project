
### Подсчёт общего количества файлов в директории и поддиректориях

Представьте, что вы — главный библиотекарь в огромной цифровой библиотеке, где книги (файлы) разложены по бесчисленным полкам и подполкам (директориям и поддиректориям). Вам срочно нужно узнать точное количество всех "книг", хранящихся в вашей главной библиотечной секции digital_library, без учёта самих полок. Эта задача кажется колоссальной, но у вас есть мощное заклинание — Files.walk().

Ваша задача — написать программу, которая, используя это заклинание, пройдёт по всей вашей библиотеке, начиная с указанной директории (например, Paths.get("digital_library")). Она должна тщательно подсчитать каждый файл, игнорируя при этом все папки. В конце, ваша программа должна гордо объявить единственное, точное число — общее количество "книг" в вашей обширной коллекции.

#### Требования:
1. Программа должна использовать метод Files.walk() для обхода всех файлов и поддиректорий, начиная с указанной директории.
2. В процессе обхода должны учитываться только файлы, а директории должны игнорироваться при подсчёте.
3. Обход файловой системы должен начинаться с указанной директории (например, Paths.get("digital_library")).
4. В конце работы программа должна вывести на экран только одно число — общее количество найденных файлов.
5. Программа должна корректно обрабатывать возможные исключения, возникающие при работе с файловой системой (например, IOException).
