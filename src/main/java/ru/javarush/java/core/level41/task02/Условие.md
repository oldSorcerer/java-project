
### Хроника чтения древних манускриптов ⏱️

Ваша роль — научный хронометрист в лаборатории по изучению древних цифровых манускриптов. Один такой манускрипт называется "input.txt". Ваша миссия не только в том, чтобы дотошно прочитать его байт за байтом с помощью FileInputStream, как это делал ваш коллега-библиотекарь в предыдущей задаче, но и тщательно зафиксировать, сколько времени на это уходит.

Время — бесценный ресурс, и вы должны измерить его в миллисендах, чтобы понять, насколько эффективен этот "побайтовый" метод. По завершении вашей кропотливой работы программа должна гордо объявить две вещи: сначала общее количество байтов, которые удалось прочитать ("Прочитано байт: X"), а затем — затраченное на это драгоценное время ("Время чтения: Y мс").

#### Требования:
1. Программа должна открывать файл "input.txt" для чтения с помощью класса FileInputStream.
2. Файл должен быть прочитан по одному байту за раз, без использования буферизации или чтения блоками.
3. Программа должна корректно подсчитать и вывести общее количество байтов, считанных из файла.
4. Программа должна измерять и сохранять время, затраченное на чтение файла, в миллисекундах.
5. Сначала на экран должно быть выведено сообщение "Прочитано байт: X" (где X — количество прочитанных байтов), затем — "Время чтения: Y мс" (где Y — время в миллисекундах).
6. FileInputStream должен быть закрыт после завершения чтения файла.
