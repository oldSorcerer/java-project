
### Запись великой истины в анналы истории ✍️

Вы — летописец, и вам доверено запечатлеть одну из самых важных истин нашей эпохи: "Java is awesome!". Это знание должно быть сохранено для потомков в нетронутом виде.

Напишите Java-программу, которая выступит в роли вашего верного летописца. Она должна создать совершенно новый текстовый фолиант, "анналы_истины.txt" (ваш output.txt). Используя высокоточный инструмент BufferedWriter, программа должна аккуратно и точно записать в этот фолиант вашу великую истину. После того как программа завершит свою работу, "анналы_истины.txt" должен содержать ровно одну строку: "Java is awesome!", без каких-либо лишних символов или добавлений.

#### Требования:
1. Программа должна использовать класс BufferedWriter для записи текста в файл.
2. Программа должна создать новый файл с именем "анналы_истины.txt" (или "output.txt" по требованию).
3. В файл должна быть записана ровно одна строка: "Java is awesome!", без лишних символов, пробелов или дополнительных строк.
4. BufferedWriter должен быть корректно закрыт после завершения записи.
5. В файле не должно быть никаких других строк, пустых строк или символов кроме "Java is awesome!".
