
### Подсчёт энергетического потенциала артефакта ⚛️

Вы — исследователь древних артефактов, и перед вами бинарный файл "энергетический_артефакт.bin". Легенда гласит, что его истинная сила (или энергетический потенциал) заключена в сумме всех числовых значений его байтов. Каждый байт в этом файле представляет собой число от 0 до 255.

Напишите Java-программу, которая выступит в роли вашего анализатора. Она должна прочитать всё содержимое файла "энергетический_артефакт.bin" с помощью FileInputStream, словно сканируя артефакт на предмет его внутренних энергий. Ваша задача — вычислить общую сумму значений всех прочитанных байтов и вывести эту сумму на экран. Если, к вашему разочарованию, файл "энергетический_артефакт.bin" окажется пустым, программа должна сообщить, что его энергетический потенциал равен 0.

#### Требования:
1. Программа должна использовать класс FileInputStream для чтения данных из файла "энергетический_артефакт.bin".
2. Программа должна последовательно прочитать все байты из файла "энергетический_артефакт.bin" до достижения конца файла.
3. Каждый считанный байт должен рассматриваться как число от 0 до 255 (использовать int value = byteValue & 0xFF или аналогичный способ).
4. Программа должна вычислить сумму всех числовых значений считанных байтов.
5. Если файл содержит байты, программа должна вывести вычисленную сумму на экран.
6. Если файл пустой (не содержит ни одного байта), программа должна вывести сообщение, что энергетический потенциал равен 0.
7. Программа должна корректно закрывать FileInputStream после завершения чтения данных.
