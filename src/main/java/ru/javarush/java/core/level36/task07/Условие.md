
### Кодирование и декодирование секретного послания ✉️

Вы — криптограф, и вам нужно передать секретное послание, состоящее из различных типов информации, так, чтобы она оставалась упорядоченной и легко читаемой только для получателя.

Ваше послание состоит из трёх частей: кодовое число, десятичный идентификатор и булево подтверждение. Напишите Java-программу, которая сначала закодирует эти данные. Она должна взять целое число 100, число с плавающей точкой 3.14 и булево значение true, а затем аккуратно записать их в бинарный файл "секретное_послание.bin" с помощью DataOutputStream.

Сразу после записи ваша программа должна декодировать это послание. Используя DataInputStream, она должна прочитать эти три значения из файла в том же самом порядке, в котором они были записаны, а затем вывести их на экран, чтобы подтвердить, что сообщение успешно передано и расшифровано.

#### Требования:
1. Программа должна использовать класс DataOutputStream для записи данных в бинарный файл.
2. Программа должна записывать в файл три значения в следующем порядке: сначала целое число 100, затем число с плавающей точкой 3.14, затем булево значение true.
3. Программа должна использовать класс DataInputStream для чтения данных из бинарного файла.
4. Программа должна читать значения из файла в том же порядке, в каком они были записаны: целое число, число с плавающей точкой, булево значение.
5. После чтения всех трёх значений программа должна вывести их на экран, чтобы подтвердить успешную передачу и расшифровку сообщения.
6. Программа должна использовать файл с именем "секретное_послание.bin" для записи и последующего чтения данных.
