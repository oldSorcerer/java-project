
### Закрепление послания в камне с магической печатью ✨

Вы — древний маг, и вам нужно высечь важное послание "Hello, Java!" на магическом камне (вашем файле output.txt). Но самое главное — после того, как послание будет высечено, магический портал, через который вы взаимодействуете с камнем (файловый ресурс), должен автоматически и безопасно закрыться, чтобы избежать утечки энергии.

Напишите Java-программу, которая выполнит эту задачу. Она должна открыть файл "магический_камень.txt" для записи, используя BufferedWriter внутри особой магической конструкции — try-with-resources. Именно эта конструкция гарантирует, что портал закроется сам по себе, даже если что-то пойдёт не так. Ваша программа должна высечь послание "Hello, Java!" в камне. После завершения ритуала файл "магический_камень.txt" должен содержать точно эту строку.

#### Требования:
1. Программа должна использовать конструкцию try-with-resources для автоматического закрытия файлового ресурса.
2. Внутри блока try-with-resources необходимо открыть файл с именем "магический_камень.txt" для записи.
3. Для записи в файл должен использоваться объект класса BufferedWriter.
4. В файл "магический_камень.txt" должна быть записана строка "Hello, Java!".
5. После выполнения записи файловый ресурс (BufferedWriter) должен быть автоматически закрыт конструкцией try-with-resources, без явного вызова метода close().
