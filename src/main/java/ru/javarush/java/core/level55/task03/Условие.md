
### Золотой Запас Игрового Мира: Учет Достижений

Представьте, что вы — хранитель сокровищ в волшебном мире игры, и ваша задача — учитывать и объявлять о великих достижениях игроков. Когда игрок совершает нечто по-настоящему выдающееся, ему начисляется особый "Золотой Запас".

Запустите асинхронную операцию с помощью CompletableFuture.supplyAsync, которая немедленно выдаст священное число — 42 единицы Золотого Запаса. Как только эта миссия по начислению завершится, используйте механизм оповещения (thenAccept), чтобы на главном экране игрового мира появилось торжественное объявление: "Результат: 42", символизирующее получение игроком его заслуженной награды.

#### Требования:
1. В программе должна быть использована асинхронная операция, созданная с помощью CompletableFuture.supplyAsync.
2. Асинхронная задача, переданная в supplyAsync, должна возвращать число 42.
3. После завершения асинхронной задачи должен быть вызван thenAccept для обработки результата.
4. В thenAccept результат должен быть выведен на экран в формате "Результат: 42".
5. Операция начисления и последующий вывод должны выполняться асинхронно, без блокировки основного потока.
