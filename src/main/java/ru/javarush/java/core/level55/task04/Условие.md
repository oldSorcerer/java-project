
### Дешифровка Секретного Сообщения: Цепочка Разведки

Вы — главный дешифровщик в секретной организации, и на ваш стол легло особо важное, зашифрованное сообщение. Ваша задача — не просто расшифровать его, но и проследить весь путь обработки информации, шаг за шагом.

Начните эту сложную операцию, асинхронно получив исходную зашифрованную строку, например, "Java Future", используя CompletableFuture.supplyAsync. Затем, как только вы получите этот фрагмент, немедленно подключите первый этап анализа (thenApply), чтобы определить точную длину этого послания. После определения длины, передайте эту информацию второму этапу (thenAccept), который немедленно выведет на экран точное досье, например: "Длина строки: X", где X — это вычисленное значение. И, наконец, чтобы подтвердить полную успешность всей операции дешифровки, запустите финальный протокол (thenRun), который торжественно объявит: "Обработка завершена!".

#### Требования:
1. Необходимо использовать CompletableFuture.supplyAsync для асинхронного получения исходной зашифрованной строки.
2. После получения строки необходимо применить thenApply для вычисления длины этой строки.
3. Вычисленную длину строки нужно передать в thenAccept, который должен вывести на экран сообщение в формате "Длина строки: X", где X — число символов.
4. После вывода длины строки необходимо с помощью thenRun вывести на экран сообщение "Обработка завершена!".
5. Цепочка CompletableFuture должна быть выстроена так, чтобы этапы выполнялись строго друг за другом: получение строки → вычисление длины → вывод длины → финальное сообщение.
