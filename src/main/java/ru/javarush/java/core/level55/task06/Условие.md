
### Сборка Приветствия: Объединение Фрагментов Речи

Вы работаете над продвинутым чат-ботом, который собирает приветствия из разных источников, чтобы сделать их более динамичными. Сейчас вам нужно объединить две части одного приветствия, которые приходят асинхронно.

Сформируйте первую асинхронную задачу CompletableFuture, которая вернет начальную часть приветствия, например, строку "Hello". Одновременно с этим, запустите вторую асинхронную задачу, которая предоставит вторую часть — строку "World". Затем, используйте специальный "склеивающий" механизм thenCombine, который возьмет обе эти части и соединит их в одну цельную, красивую фразу, например, "Hello, World!". Как только это полное приветствие будет готово, выведите его на экран, чтобы ваш чат-бот мог порадовать пользователя.

#### Требования:
1. Программа должна создать два отдельных объекта CompletableFuture, каждый из которых асинхронно возвращает строку ("Hello" и "World" соответственно).
2. Для объединения результатов двух CompletableFuture должен быть использован метод thenCombine.
3. Метод thenCombine должен объединять две строки в одну фразу с запятой и пробелом между ними (например, "Hello, World!").
4. Обе части приветствия должны запускаться параллельно, а не последовательно, чтобы использовать преимущества асинхронности.
5. Как только объединённая строка будет готова, программа должна вывести её на экран.
