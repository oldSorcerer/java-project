
### Координация Поставок: Ожидание Всех Грузов

Представьте, что вы управляете огромным логистическим центром, и на ваш склад должны прибыть три очень важных, но независимых груза. Ваша главная задача — убедиться, что все три груза доставлены, прежде чем вы сможете выдать сводный отчет.

Создайте три независимые асинхронные задачи CompletableFuture, каждая из которых символизирует прибытие одного из грузов, возвращая уникальное обозначение, например, "A", "B" и "C". Как только вы инициируете отправку всех трех грузов, используйте централизованную систему отслеживания allOf, чтобы терпеливо дождаться завершения доставки каждого из них. Только после того, как все три груза будут на складе, вы сможете вывести на экран их полные идентификаторы, разделенные пробелом, например: A B C.

#### Требования:
1. Необходимо создать три объекта CompletableFuture, каждый из которых выполняет независимую асинхронную задачу и возвращает строку с уникальным идентификатором ("A", "B" и "C").
2. Каждая из трех задач должна быть запущена асинхронно, чтобы выполняться независимо друг от друга.
3. Для ожидания завершения всех трех асинхронных задач необходимо использовать метод CompletableFuture.allOf.
4. Программа должна дожидаться завершения всех трех CompletableFuture перед выполнением дальнейших действий (например, с помощью метода join или get).
5. После завершения всех задач необходимо получить результат (строку-идентификатор) из каждого CompletableFuture.
6. Результаты всех трех задач должны быть выведены в одну строку, разделённые пробелом, в формате: "A B C".
7. Каждая асинхронная задача должна возвращать свой уникальный идентификатор ("A", "B" или "C"), а не одинаковое значение.
