
### Сердце Мыслителя: Асинхронная Трансформация Энергии

Представьте, что вы программируете главный процессор футуристического города, который питается от особого "Мыслителя". Ваша задача — асинхронно рассчитать энергетический квадрат ключевого значения, а затем представить результат в удобном для горожан формате.

Для начала, представьте, что есть некое базовое энергетическое значение, например, 8. Используйте CompletableFuture, чтобы в фоновом режиме, не блокируя основную работу города, вычислить квадрат этого числа. Как только расчет будет завершен, примените специальный преобразователь (thenApply), чтобы превратить это числовое значение в читаемую строку, например, "Результат: 64".

После этого, задействуйте систему отображения (thenAccept), чтобы эта важная информация немедленно появилась на главном экране города. Не забудьте, как только задача по расчету будет инициирована, мгновенно вывести сообщение "Вычисление запущено!", чтобы все знали, что Мыслитель начал свою работу.

#### Требования:
1. В задаче должен быть использован класс CompletableFuture для выполнения асинхронных вычислений.
2. Необходимо инициировать асинхронную задачу, которая вычисляет квадрат заданного числа (например, 8).
3. После получения результата вычисления квадрата, к нему должен быть применён метод thenApply для преобразования числа в строку формата "Результат: <значение>".
4. К преобразованному строковому результату должен быть применён thenAccept, который выводит полученную строку на экран.
5. Сразу после запуска асинхронной задачи, до получения результата, программа должна вывести на экран сообщение "Вычисление запущено!".
