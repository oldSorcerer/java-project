
### Извлечение файла из ZIP-архива

Ваша миссия продолжается! Теперь вам нужно извлечь то самое сверхсекретное послание, которое вы упаковали в архив.

Реализуйте программу, которая "вскроет" архив "hello.zip", найдёт внутри него файл "hello.txt" и аккуратно извлечёт его содержимое, сохранив его в обычный текстовый файл "output.txt" в текущей директории.

Используйте ZipInputStream, чтобы безопасно получить доступ к вашей ценной информации.

#### Требования:
1. Программа должна использовать класс ZipInputStream для работы с ZIP-архивом.
2. Программа должна открывать архив с именем "hello.zip" в текущей директории.
3. Программа должна находить внутри архива файл с именем "hello.txt".
4. Программа должна извлекать содержимое файла "hello.txt" из архива.
5. Извлечённое содержимое должно быть сохранено в файл "output.txt" в текущей директории.
6. Все потоки ввода/вывода должны быть корректно закрыты после завершения работы.
