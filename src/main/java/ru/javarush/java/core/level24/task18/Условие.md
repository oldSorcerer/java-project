
### Магический артефакт: Самостоятельное отключение

Представьте, что вы управляете очень деликатным магическим артефактом, который нужно тщательно "отключать" после использования, иначе могут быть непредсказуемые последствия. Ваша цель — убедиться, что этот артефакт самостоятельно завершает работу, как только его перестают использовать, даже если в процессе работы возникнут проблемы.

Создайте класс, назовите его MagicArtifact. Этот класс должен быть умным и реализовывать специальный интерфейс AutoCloseable. В методе close() этого класса (который вызывается автоматически при закрытии) выведите сообщение: "Магический артефакт безопасно отключён".

В главной части вашей программы (методе main) используйте конструкцию try-with-resources для создания объекта MagicArtifact. Внутри блока try выведите сообщение "Идёт работа с магическим артефактом". После завершения блока try, программа должна завершить свою работу, а в консоли вы обязательно должны увидеть сообщение о том, что магический артефакт был безопасно отключён, подтверждая его способность к самостоятельному "завершению дел".

#### Требования:
1. Класс MagicArtifact должен реализовывать интерфейс AutoCloseable.
2. В классе MagicArtifact должен быть реализован метод close(), который выводит на экран сообщение: "Магический артефакт безопасно отключён".
3. В методе main объект класса MagicArtifact должен создаваться внутри конструкции try-with-resources.
4. Внутри блока try-with-resources должно выводиться сообщение: "Идёт работа с магическим артефактом".
5. После выхода из блока try-with-resources в консоли обязательно должно появиться сообщение о безопасном отключении магического артефакта.
