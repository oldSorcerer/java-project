
### Ваш Персональный Нотификатор

Представьте, что вы создаёте гибкую систему уведомлений для своего приложения. Вам нужна возможность быстро определить, как будет выводиться сообщение, без создания полноценного класса для каждого типа уведомления.

Объявите специальный "**контракт**" для вывода сообщений — интерфейс MessagePrinter. Он должен иметь ровно один абстрактный метод: void print(String message), который принимает текст сообщения. Отметьте этот интерфейс аннотацией @FunctionalInterface, чтобы показать, что он предназначен для компактных реализаций.

В основной части вашей программы создайте переменную типа MessagePrinter и присвойте ей лямбда-выражение. Это лямбда-выражение должно просто принимать переданное сообщение и выводить его на экран. Затем вызовите этот метод, передав в него строку "Привет, Java!", и убедитесь, что ваше уведомление работает.

#### Требования:
1. Интерфейс MessagePrinter должен быть объявлен с ровно одним абстрактным методом void print(String message).
2. Интерфейс MessagePrinter должен быть помечен аннотацией @FunctionalInterface.
3. В основной части программы должна быть создана переменная типа MessagePrinter.
4. Переменной типа MessagePrinter должно быть присвоено лямбда-выражение, реализующее метод print так, чтобы оно выводило полученное сообщение на экран.
5. Метод print у переменной MessagePrinter должен быть вызван с аргументом "Привет, Java!", и программа должна вывести это сообщение на экран.
