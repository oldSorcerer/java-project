
### ранспортная Компания: Единый Протокол Движения

Вы разрабатываете программное обеспечение для управления автопарком транспортной компании. Чтобы все транспортные средства, будь то легковые автомобили, грузовики или автобусы, могли получать команду "двигаться" по единому стандарту, вам необходимо установить общий протокол.

Определите такой протокол в виде интерфейса Movable (Перемещаемый), который содержит единственную команду: void move() (двигаться).

Затем создайте класс Car (Автомобиль), который будет представлять одно из ваших транспортных средств. Этот Car должен полностью соответствовать протоколу Movable. Когда вашему Car отдают команду move(), он должен гордо объявлять на экране: "Машина едет".

#### Требования:
1. Необходимо создать интерфейс с именем Movable.
2. Интерфейс Movable должен содержать единственный абстрактный метод void move() без реализации.
3. Необходимо создать класс с именем Car.
4. Класс Car должен реализовывать интерфейс Movable.
5. Класс Car должен предоставлять конкретную реализацию метода move(), определенного в интерфейсе Movable.
6. Метод move() в классе Car должен выводить на экран строку "Машина едет".
