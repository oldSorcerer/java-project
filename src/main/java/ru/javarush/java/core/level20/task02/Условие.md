
### Художник, Мастер Линий и Кругов

Вы работаете над созданием интерактивного приложения для рисования. В этом приложении каждая фигура должна знать, как себя отобразить на холсте. Вы хотите, чтобы все фигуры, независимо от их типа, можно было "попросить" нарисовать себя универсальным способом.

Сперва определите универсальный чертежный контракт под названием Drawable, который включает в себя одно действие: draw(), не возвращающее ничего. Затем создайте конкретную фигуру — Circle (Круг). Этот Circle должен чётко понимать, как выполнить контракт Drawable, и при вызове draw() он должен объявлять: "Drawing a circle".

Теперь в вашей основной программе, представьте, что у вас есть универсальный инструмент для рисования, который может удерживать любой объект, соответствующий контракту Drawable. Присвойте этому инструменту ваш только что созданный Circle. Затем прикажите вашему инструменту draw(). В результате на экране должно появиться сообщение "Drawing a circle".

#### Требования:
1. Должен быть объявлен интерфейс с именем Drawable.
2. Интерфейс Drawable должен содержать абстрактный метод draw(), который не принимает параметров и не возвращает значение (void).
3. Должен быть создан класс Circle, который реализует интерфейс Drawable.
4. В классе Circle должен быть реализован метод draw(), который выводит на экран строку "Drawing a circle".
5. В основной программе должна быть объявлена переменная типа Drawable, которой присваивается объект класса Circle.
6. Метод draw() должен быть вызван у переменной типа Drawable, и в результате на экран должно быть выведено сообщение "Drawing a circle".
