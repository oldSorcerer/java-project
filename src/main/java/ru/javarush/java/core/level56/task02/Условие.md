
### Асинхронная запись важного события в журнал ✍️

Вы разрабатываете критически важную систему логирования, которая должна оперативно фиксировать все происходящие события, не дожидаясь завершения каждой записи. Вам нужно записать важное системное сообщение в файл журнала.

Ваша миссия — открыть файл "output.txt" для записи. Если такого файла ещё нет, система должна самостоятельно его создать. Затем вы асинхронно, используя AsynchronousFileChannel и его метод write, записываете в этот файл строку "Hello, async world!". Поскольку операция выполняется асинхронно, она вернёт специальный объект Future. После того как запись завершится, вы должны получить из этого Future информацию о том, сколько именно байт было успешно записано, и вывести это число на экран. Это позволит убедиться, что ваше важное сообщение надёжно сохранено в журнале.

#### Требования:
1. Для записи в файл должен использоваться класс AsynchronousFileChannel.
2. Файл "output.txt" должен быть открыт для записи с флагами, позволяющими создать его, если он отсутствует.
3. Строка "Hello, async world!" должна быть записана в файл асинхронно с помощью метода write AsynchronousFileChannel.
4. Результат асинхронной операции должен быть получен через объект Future, возвращаемый методом write.
5. После завершения записи программа должна вывести на экран количество байт, которое было успешно записано в файл.
6. Канал AsynchronousFileChannel должен быть корректно закрыт после завершения работы.
